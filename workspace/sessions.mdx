Within this section, all sessions at **Kelva** are <u>logged</u> and <u>visible</u>. This not only includes <u>Training Sessions</u>, but also **Shifts** at the main game.

You will be able to see a **Weekly Schedule** of that current week <u>scheduled sessions</u>.

### How can I create a shift session?

To create a <u>shift session</u> successfully, follow these steps:

<Steps>
  <Step title="Creating the Session" icon="circle-plus">
    Click on "**New Session**", to create a <u>new session type</u>.
  </Step>
  <Step title="Basic Information" icon="circle-info">
    Enter the following details:

    1. **Session Name:** `Shift Session`.
    2. **Description:** `None`.
    3. **Session Type:** `Shift`.
    4. **Game ID:** `18887062846`.

    After entering these details, <u>click</u> on "**Continue to Scheduling**".
  </Step>
  <Step title="Sheduling" icon="timer">
    Select "**Unsheduled session**".

    Set the date and time for your shift:

    1. **Session Date:** `The current date`.
    2. **Session Time:** `The time in your local timezone`.

    After entering these details, <u>click</u> on "**Continue to Permissions**".
  </Step>
  <Step title="Permissions" icon="users">
    Select the following roles:

    - `Management Team`
    - `Corporate Team`
    - `Leadership Team`

    After selecting these details, <u>click</u> on "**Continue to Statuses**".
  </Step>
  <Step title="Statuses" icon="square-list">
    We will not add any status, <u>click</u> on "**Continue to Slots**".
  </Step>
  <Step title="Slots" icon="calendar-pen">
    Add a new with the following details:

    1. **Role Name:** `Staff Management Department`.
    2. **Available Slots:** `6`.

    Add another one with the following details:

    1. **Role Name:** `Public Relations Department`.
    2. **Available Slots:** `6`.

    After entering these details, <u>click</u> on "**Create Session**".
  </Step>
</Steps>

# All done!

<Card title="Logging" icon="pencil">
  The **session** has now been **created** successfully. Make sure to **log** <u>all</u> staff members that attend, as well as the <u>Host</u> (you), and the <u>Co-Host</u>.
</Card>
